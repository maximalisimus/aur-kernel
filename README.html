Описание

В данном репозитории находятся старые версии ядер LTS с долгосрочной поддержкой для Archlinux.

Подключение

Для добавления репозитория в систему отредактируйте файл: /etc/pacman.conf
Обратите внимание на строку SigLevel. Все коммиты данного репозитория верифицированы цифровой подписью. Поэтому выполнять проверку пакетов в "pacman" необязательно. 

[aur-kernel]
SigLevel = Never
Server = https://maximalisimus.github.io/$repo/os/$arch

Однако вы можете отказаться от строки SigLevel, но для этого перед добавлением репозитория в конфигурацию pacman установите публичные ключи доступа.

$ wget https://maximalisimus.github.io/elseworld-keyring/install-keyring.sh
$ sudo sh install-keyring.sh

Или загрузите и установите публичные ключи доступа вручную следующими командами.

$ sudo pacman-key --init
$ wget https://maximalisimus.github.io/elseworld-keyring/elseworld{.gpg,-revoked,-trusted,-id.txt}
$ sudo mv elseworld{.gpg,-revoked,-trusted} /usr/share/pacman/keyrings/
$ sudo pacman-key --add /usr/share/pacman/keyrings/elseworld.gpg
$ id_keyring=$(cat elseworld-id.txt)
$ sudo pacman-key --lsign-key ${id_keyring}
$ sudo pacman-key --populate archlinux elseworld
$ rm -rf elseworld-id.txt

Подробная информация о пакетах данного репозитория

Репозиторий с публичными ключами:Release-keyring
md5sums:md5sums.txt
Контроль версий (минимально):version.txt
Контроль версий (подробно):Release.html

Файл version.txt представляет из себя текстовый документ, в который выведена минимальная информация о пакетах данного репозитория.
Предусмотрена возможность работы с файлом через консоль.

$ cat version.txt
$ cat version.txt | grep -Evi "^#" | sort | grep -Ei "linux-lts316" | awk '{print $1}'
	$ linux-lts316
$ cat version.txt | grep -Evi "^#" | sort | grep -Ei "linux-lts316" | awk '{print $2}'
	$ 3.16.85-1

Аналогичный поиск минимальной информации по какому-либо пакету можно осуществить через pacman.

$ pacman -Ss | grep -Ei "^community/" | sed 's/community\///g' | grep -Ei "supertuxkart" | awk '{print $1}'
	$ supertuxkart
$ pacman -Ss | grep -Ei "^community/" | sed 's/community\///g' | grep -Ei "supertuxkart" | awk '{print $2}'
	$ 1.1-2

Файл Release.html представляет из себя веб-страницу с подробной информацией о каждом пакете данного репозитория в табличной форме.
Страница поддерживает строку поиска.

Description

This repository contains older versions of LTS cores with long-term support for Archlinux.

Connection

To add a repository to the system, edit the file: /etc/pacman.conf
Note the SigLevel string. All commits in this repository are digitally signed. Therefore, it is not necessary to check packages in "pacman". 

[aur-kernel]
SigLevel = Never
Server = https://maximalisimus.github.io/$repo/os/$arch

However, you can opt out of the SigLevel string, but set public access keys before adding the repository to the pacman configuration.

$ wget https://maximalisimus.github.io/elseworld-keyring/install-keyring.sh
$ sudo sh install-keyring.sh

Or download and install public access keys manually using the following commands.

$ sudo pacman-key --init
$ wget https://maximalisimus.github.io/elseworld-keyring/elseworld{.gpg,-revoked,-trusted,-id.txt}
$ sudo mv elseworld{.gpg,-revoked,-trusted} /usr/share/pacman/keyrings/
$ sudo pacman-key --add /usr/share/pacman/keyrings/elseworld.gpg
$ id_keyring=$(cat elseworld-id.txt)
$ sudo pacman-key --lsign-key ${id_keyring}
$ sudo pacman-key --populate archlinux elseworld
$ rm -rf elseworld-id.txt

$ cat version.txt
