Подробная информация о каждом пакете данного репозитория
(Detailed information about each package in this repository).

Last Updated - Дата последней сборки пакета в данном репозитории.
Date when the package was last built in this repository.

Depends - Массив имен пакетов, которые должны быть установлены для запуска программы из данного пакета.
An array of packages that must be installed for the software to build and run.

Optdepends - Массив имен пакетов, которые не требуются для работы программы, но предлагают дополнительные функции. Это может означать, что не все исполняемые файлы, предоставляемые пакетом, будут работать без соответствующих дополнительных зависимостей.
An array of packages that are not needed for the software to function, but provide additional features. This may imply that not all executables provided by a package will function without the respective optdepends.

Makedepends - Массив имен пакетов, которые должны быть установлены лишь для сборки пакета, но не нужны для использования программы после установки.
An array of packages that are only required to build the software.
